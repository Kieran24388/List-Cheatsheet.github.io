HTML Cheatsheet - Lists

Types of list in HTML

Ordered Lists
A list where each item in the list is numbered eg. steps in a recipe.
Unordered Lists
A list where each item has a bullet point.
Definition Lists
A list made up of a set of terms along with the definition of each of those terms. This list is an example of a definition list.

HTML list elements

Tag Name Description Notes
<ol> Ordered list Used to create an ordered list Each item is placed within an opening <li> tag and a closing </li> tag. Each item will be preceded by a sequential number.
<ul> Unordered list Used to create an unordered list Each item is placed within an opening <li> tag and a closing </li> tag. Each item will be prededed by a bullet point.
<li> List item Used as a child of <ol> or <ul> to enclose list items Browsers indent list items by default.
<dl> Definition list Used to list a series of terms and their definitions eg. a dictionary Example of definition list shown above in 'Types of list in HTML' section. Inside the <dl> element you will usually see pairs of <dt> and <dd> elements.
<dt> Definition term Used to contain the term being defined Usually a child of the <dl> element. Example: a word in the dictionary.
<dd> Definition Used to contain the definition Provides a definition for the term in the <dt> element.
Additional notes:
  • Lists can also be nested inside other lists. The browser will render these with a further indentation and usually with a different style of bullet point.
